# BukkitItemBuilder
Useful item builder library for Bukkit plugins.

```gradle
repositories {
    mavenCentral()
}

dependencies {
    implementation("io.github.portlek:BukkitItemBuilder:1.0")
}
```

```xml
<dependecies>
    <depedency>
        <groupId>io.github.portlek</groupId>
        <artifactId>BukkitItemBuilder</artifactId>
        <version>1.0</version>
    </depedency>
</dependecies>
```
