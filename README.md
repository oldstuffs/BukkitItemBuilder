## Setup

<details>
<summary>Gradle</summary>

```gradle
repositories {
    mavenCentral()
}

dependencies {
    implementation("io.github.portlek:BukkitItemBuilder:${version}")
}
```
</details>

<details>
<summary>Maven</summary>

```xml
<dependencies>
    <dependency>
      <groupId>io.github.portlek</groupId>
      <artifactId>BukkitItemBuilder</artifactId>
      <version>${version}</version>
    </dependency>
</dependencies>
```
</details>
